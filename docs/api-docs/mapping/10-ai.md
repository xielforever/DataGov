# AI 助手页面映射

## 10.1 全局 AI Copilot

- **组件**：`src/components/ai/AiAssistant.tsx`
- **入口**：登录后全局右下角浮层，快捷键 `Alt+A`
- **页面上下文**：当前菜单 `viewId`、页面标题 `viewTitle`、URL、SQL 方言、用户选中内容和页面上下文块

## 10.2 前端交互与接口

| UI 能力 | 接口 |
| --- | --- |
| 能力卡片 | `GET /api/v1/ai/capabilities` |
| 会话列表/搜索 | `GET /api/v1/ai/conversations` |
| 新建会话 | `POST /api/v1/ai/conversations` |
| 会话详情 | `GET /api/v1/ai/conversations/{id}` |
| 收藏/归档会话 | `PATCH /api/v1/ai/conversations/{id}` |
| 发送消息 | `POST /api/v1/ai/conversations/{id}/messages` |
| 重新生成 | `POST /api/v1/ai/messages/{id}/regenerate` |
| 点赞/点踩 | `POST /api/v1/ai/messages/{id}/feedback` |
| 用户行为事件 | `POST /api/v1/ai/behavior-events` |
| 用户偏好 | `GET /api/v1/ai/preferences`、`PUT /api/v1/ai/preferences` |
| 上下文预览 | `POST /api/v1/ai/context/preview` |
| Token 用量 | `GET /api/v1/ai/token-usage` |
| 只读工具列表 | `GET /api/v1/ai/tools` |

## 10.3 当前实现状态

- 前端 AI 面板已支持会话历史、搜索、收藏、归档、Token 预览、用户偏好、反馈和行为事件。
- 后端 Go AI Gateway 持久化会话、消息、反馈、偏好、上下文快照、Token 用量和工具调用审计。
- 默认真实后端接入 MiniMax Anthropic-compatible API；仅在 `VITE_REAL_AI_ASSISTANT=false` 时由 MSW 兜底。
- 首期工具均为只读治理工具，不直接访问外部源数据。
